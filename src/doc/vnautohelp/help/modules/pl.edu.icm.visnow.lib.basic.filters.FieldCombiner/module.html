Combine Field Components

 

         

The module creates a new output field of all components of all input fields. The output field has properties (field dimensions, geometric extent, physical extent, origin, cell vectors) of the field which was connected first. Fields with other dimensions are not processed.

Input data

Any fields.

Output data

At output there is the combined field.
